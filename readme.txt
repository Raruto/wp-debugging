A support/troubleshooting plugin for WordPress.

## Description

This plugin sets many of the WordPress debug constants to `true`. Additionally it installs and activates the [Query Monitor](https://wordpress.org/plugins/query-monitor/) and [Debug Quick Look](https://github.com/norcross/debug-quick-look) plugins to aid in debugging and troubleshooting.
